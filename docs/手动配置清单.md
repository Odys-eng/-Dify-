## 快速启动检查清单

> 按以下顺序操作，从零开始到运行成功。

### Phase 0：环境准备（约 30 分钟）

```
☐ 0.1  安装 Docker Desktop（含 WSL2 后端）
        验证：docker version + docker compose version

☐ 0.2  安装 Git
        验证：git --version

☐ 0.3  安装 Python 3.10+
        验证：python --version

☐ 0.4  安装 Python 依赖
        pip install requests pypdf2
        验证：python -c "import requests, PyPDF2"

☐ 0.5  确认 80 端口未被占用
        验证：netstat -ano | findstr :80（无输出或可释放）
```

### Phase 1：申请 API Key（约 10 分钟）

```
☐ 1.1  注册 DeepSeek 平台 → 获取 API Key
        网址：https://platform.deepseek.com
        验证：curl 测试返回 200

☐ 1.2  注册 Tavily 平台 → 获取 API Key
        网址：https://tavily.com
        验证：curl 测试返回 200

☐ 1.3  ⚠️ 确认 DeepSeek 模型名为 deepseek-v4-flash
        （不是 deepseek-chat，后者 2026-07-24 停用）
```

### Phase 2：准备知识库数据（约 20 分钟）

```
☐ 2.1  下载 FANUC 0i-MF 维修手册 PDF（公开版）
        存放：data/pdfs/

☐ 2.2  下载 Siemens 840D sl 操作手册 PDF（公开版）
        存放：data/pdfs/

☐ 2.3  下载/整理 CNC 故障案例集 PDF
        存放：data/pdfs/

☐ 2.4  确认 PDF 为文字版（可复制文字，非扫描件）
        验证：用 PDF 阅读器选中文字，能复制
```

### Phase 3：启动 Dify（约 10 分钟）

```
☐ 3.1  复制环境变量模板
        cp deploy/.env.template dify/docker/.env

☐ 3.2  编辑 .env，填入：
        DEEPSEEK_API_KEY=你的实际Key
        TAVILY_API_KEY=你的实际Key

☐ 3.3  运行启动脚本
        Linux/Mac：./deploy/startup.sh
        Windows：.\deploy\startup.ps1

☐ 3.4  等待服务启动（约 1-2 分钟）
        验证：docker compose ps 全部 running

☐ 3.5  运行环境验证
        ./deploy/verify-setup.sh
        验证：4 项检查全部通过
```

### Phase 4：Dify 控制台配置（约 40 分钟）

```
☐ 4.1  浏览器访问 http://localhost → 创建管理员账号

☐ 4.2  设置 → 模型供应商 → DeepSeek
        API Key：粘贴 DeepSeek Key
        模型：deepseek-v4-flash
        验证：模型测试返回正常回复

☐ 4.3  设置 → 模型供应商 → BGE → 启用 bge-large-zh-v1.5

☐ 4.4  知识库 → 创建知识库
        上传 3 份 PDF
        切片：chunk_size=500, overlap=50, 按章节
        Embedding：bge-large-zh-v1.5
        检索：top_k=3, score_threshold=0.5, Rerank 开启
        验证：召回测试返回相关片段

☐ 4.5  工具 → 自定义 → 创建 Tavily 工具
        粘贴 OpenAPI Schema
        填入 Tavily API Key
        验证：工具测试返回搜索结果

☐ 4.6  创建应用 → 从 DSL 导入 → 上传 src/workflow/chatflow-dsl.yml

☐ 4.7  Workflow 编辑器中：
        KR 节点 → 选择知识库
        Tool 节点 → 选择 Tavily 工具
        LLM 节点 → 粘贴 System Prompt（从 src/workflow/system-prompt.txt 复制）

☐ 4.8  点击「发布」
        验证：应用状态为「已发布」

☐ 4.9  应用 → API 访问 → 创建 API Key（复制保存）
        知识库 → API → 创建 API Key（复制保存）
        记录 Dataset ID（知识库 URL 中的 ID）
```

### Phase 5：验证测试（约 15 分钟）

```
☐ 5.1  知识库检索验证
        python src/knowledge/verify_retrieval.py \
          --dataset_id 你的Dataset_ID \
          --api_key dataset-xxxxxxxx
        验证：命中率 ≥ 70%

☐ 5.2  自动化验收测试
        python src/tests/test_scenarios.py \
          --api_key app-xxxxxxxx
```
